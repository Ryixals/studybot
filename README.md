# Study Bot

A Discord bot that tracks study time across different subjects, maintains progress dashboards, and displays leaderboards.

## Features

- Track study time per subject (English, Math, Science, History)
- Remove incorrect or over-reported study time
- View personal progress dashboard with visual progress bars
- Global leaderboard showing top students
- Persistent database storage for all user data

## Commands

1. /add [subject] [minutes] - Add study time to a subject (1-720 minutes)
2. /remove [subject] [minutes] - Remove study time from a subject
3. /check - Display your study progress dashboard
4. /leaderboard - Show global leaderboard of top students

## Setup

1. Clone the repository
2. Install dependencies
3. Create a `.env` file
4. Run bot.py
